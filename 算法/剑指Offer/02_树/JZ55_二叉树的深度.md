```java
    int val = 0;

    public TreeNode(int val) {
        this.val = val;
    }
}
```

## 3、代码实现

```java
public class Solution {
    
    public int TreeDepth(TreeNode root) {
        return root == null ? 0 : Math.max(TreeDepth(root.left), TreeDepth(root.right)) + 1;
    }
}
```
